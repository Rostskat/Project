# Vioty - Ice Cream Landing Page

## Опис проєкту

Vioty - це односторінковий веб-сайт (landing page) для магазину морозива. Проєкт створено як навчальний мініпроект для демонстрації базових навичок HTML5 та структурування веб-сторінок.

## Особливості

Сайт містить наступні секції:

- **Головна секція (Hero)** - привітальний екран з основним повідомленням та закликом до дії
- **Про нас (About Us)** - інформація про компанію з візуальними елементами
- **Наші продукти (Our Product)** - каталог морозива з цінами:
  - Yellow ice cream - $19.55
  - Green ice cream - $19.55
  - Pink ice cream - $19.55
  - Red ice cream - $19.55
- **Додаток (App)** - простий процес замовлення в 3 кроки
- **Форма зворотного зв'язку** - можливість поставити питання
- **Футер** - контактна інформація та навігація

## Технології

- **HTML5** - структура та розмітка сторінки
- **CSS** - стилізація (файл style.css)
- Семантичні теги HTML5 (`<header>`, `<main>`, `<section>`, `<nav>`, `<footer>`)

## Як запустити

1. Завантажте всі файли проєкту
2. Переконайтеся, що структура папок збережена (особливо папка `images/`)
3. Відкрийте файл `index.html` у веб-браузері
